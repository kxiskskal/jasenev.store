Jasenev Store

Опис: Це веб-сайт для демонстрації каталогу товарів інтернет-магазину Jasenev Store. Проєкт створено з використанням чистого HTML та CSS, він демонструє повну структуру багатосторінкового сайту, включаючи каталог, категорії та сторінки окремих товарів.

# Реалізований функціонал

У поточній версії проєкту було виконано значну роботу над фронтенд-частиною:

* **Сучасна верстка:** Впроваджено **Flexbox** (для хедерів, навігації) та **CSS Grid** (для сітки товарів) для створення стабільних адаптивних макетів.
* **Адаптивний дизайн:** Реалізовано повноцінну підтримку мобільних пристроїв за допомогою медіа-запитів (`@media`). Додано функціональне **бургер-меню** для зручної навігації на смартфонах.
* **Архітектура CSS:** Створено модульну структуру стилів (розділення на `base.css`, `layout.css`, `components.css`), що забезпечує чистоту коду та кросбраузерну сумісність без необхідності додаткових виправлень.
* **Форми та UX:** Застосовано сучасні підходи до стилізації полів вводу та кнопок. Реалізовано **HTML5-валідацію** та кастомну JavaScript-валідацію з інтерактивними повідомленнями про помилки для покращення користувацького досвіду (UX).

# Примітки / Плани на майбутнє

* Частково реалізовані сторінки (Кошик, Вхід) будуть доповнені динамічним функціоналом за допомогою JS та підключення бази даних у майбутньому.
* Інтерактивність елементів інтерфейсу буде покращуватися в наступних ітераціях через розширення JavaScript-логіки.
