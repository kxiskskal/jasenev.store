Jasenev Store

Опис: Це веб-сайт для демонстрації каталогу товарів інтернет-магазину Jasenev Store. Проєкт створено з використанням чистого HTML та CSS, він демонструє повну структуру багатосторінкового сайту, включаючи каталог, категорії та сторінки окремих товарів.

## Реалізований функціонал

### Інтерфейс та Дизайн (UI/UX)
* **Адаптивність:** Повна підтримка мобільних пристроїв, планшетів та десктопів.
* **Бургер-меню:** Кастомне меню для мобільних версій без використання бібліотек.
* **Dark Mode :**
  * Перемикач теми (Світла/Темна).
  * Збереження вибору користувача в `localStorage`.
  * Стилізація форм та компонентів під темну тему.
* **Стиль Balenciaga/Brutalism:** Агресивний дизайн цінників та знижок.
* **Accessibility:** Функція зміни розміру шрифту клавішами `ArrowUp` / `ArrowDown` з візуальним сповіщенням.

### Форми та Валідація
* **Універсальна обробка форм:** Єдиний JS-модуль для реєстрації, входу та контактів.
* **Валідація:** Перевірка email (Regex), довжини пароля, збігу паролів, обов'язкових полів.
* **Інтерактивність:**
  * Відображення помилок під конкретними полями.
  * Перемикач видимості пароля ("око").
  * Анімація успішної реєстрації (Checkmark animation) та автоперехід.
* **Інтеграція:** Контактна форма налаштована на відправку даних через сервіс **Formspree**.

### Додатково
* **Живий футер:** Відображення поточної дати та часу в реальному часі.
* **Модульна структура:** JavaScript код розділений на логічні модулі (`forms.js`, `ui.js`, `theme.js` тощо).
